# 提示词模板库

> 收集整理常用提示词模板，按场景分类。

---

## 一、写作类

### 文章撰写

```markdown
你是一位专业的 [领域] 内容创作者。
请根据以下要求撰写一篇文章：

主题：[主题]
目标读者：[读者描述]
字数：[字数要求]
风格：[正式/轻松/专业]
关键词：[关键词列表]

要求：
1. [要求 1]
2. [要求 2]

请以 [格式] 输出。
```

## 二、编程类

### 代码审查

```markdown
你是一位资深 [语言] 工程师。
请审查以下代码，关注：
1. 代码质量和可读性
2. 潜在的错误和漏洞
3. 性能优化机会
4. 最佳实践遵循情况

代码：
```[语言]
[代码内容]
```

请按以下格式输出审查意见：
- 问题类型：[严重性]
- 位置：[行号]
- 建议：[修改建议]
```

## 三、分析类

### 文档总结

```markdown
请总结以下文档，输出格式：

## 核心要点
- [3-5 个关键点]

## 详细摘要
[2-3 段]

## 行动项
- [如果有需要后续操作的事项]

原文：
[文档内容]
```
