## 指标结果

| 指标 | 数值 | 说明 |
|------|------|------|
| 格式遵循率 | 95% | 结构化 System Prompt 下输出格式正确率 |
| Few-shot 提升幅度 | +22% | 翻译一致性评分提升 |
| CoT 推理准确率提升 | +35% (GPT-4o) / +18% (Claude) | 数学推理任务 |
| CoT 对推理模型提升 | +2% | o3 已内置推理，显式 CoT 几乎无效 |

## 成功情况

1. 结构化 System Prompt 在所有测试模型上均有效，格式遵循率 >90%
2. Few-shot 示例在翻译、摘要等任务上持续有效
3. CoT 在非推理模型上显著提升推理准确率

## 失败情况

1. 显式 CoT 提示对 o3 等内置推理模型无显著额外提升
2. Few-shot 示例在 Claude 上可能导致过度遵循示例风格而损失灵活性
3. 部分"技巧"（如"take a deep breath"）效果不稳定，因模型版本而异

## 成本与延迟

| 指标 | 数值 |
|------|------|
| 单次测试平均延迟 | GPT-4o: 2.3s / Claude: 2.8s / o3: 5.1s |
| 测试总 Token 消耗 | ~15,000 tokens |
| 估算成本 | ~$0.08 (GPT-4o + Claude + o3) |

## 风险与限制

1. 提示词效果因模型版本而异，跨模型通用性有限
2. 推理模型（o 系列）与传统模型的提示词策略差异大，需分别对待
3. 部分技巧缺乏严格对照实验，效果评估可能有主观偏差
4. 本验真基于 2026-05 的模型版本，后续模型更新可能改变效果
