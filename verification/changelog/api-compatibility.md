## 变更分类说明

| 分类 | 说明 | 文档更新优先级 |
|------|------|--------------|
| **新模型** | 新模型上线 | 高 — 需立即更新模型列表和定价 |
| **新接口** | 新 API 接口发布 | 高 — 需更新代码示例 |
| **接口变更** | 现有接口参数或行为变更 | 中 — 需检查兼容性 |
| **功能扩展** | 新增可选功能 | 中 — 补充文档即可 |
| **价格变更** | 定价调整 | 高 — 需更新定价表 |
| **废弃** | 功能或参数废弃 | 高 — 需标注替代方案 |
