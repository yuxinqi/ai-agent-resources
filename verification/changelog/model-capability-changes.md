# 模型能力变更日志

> 记录各 AI 模型的能力变更，包括新能力上线、能力退化、基准测试结果更新等。

## 2026-05

### 推理能力

| 模型 | 变更 | 评估 | 说明 |
|------|------|------|------|
| o3 | 新增可视化推理能力 | 提升 | 支持图表、图形的推理分析 |

| DeepSeek-R1 | 推理一致性改善 | 提升 | 长链推理的幻觉率下降约 15% |

### 代码能力

| 模型 | 变更 | 评估 | 说明 |

| Gemini 2.5 Pro | 多文件编辑能力 | 新增 | 支持跨文件代码修改建议 |

## 能力评估指标说明

| 指标 | 说明 | 评估方法 |
|------|------|---------|
| **推理准确率** | 数学/逻辑推理题的正确率 | 标准测试集（MATH-500, GPQA） |
| **代码生成** | 代码生成的正确率和可用性 | SWE-bench, HumanEval |
| **工具调用** | Function Calling 的成功率 | 自建测试集 |
| **多模态** | 图像/音频/视频理解能力 | MMMU, 自建测试 |
| **中文能力** | 中文理解和生成质量 | C-Eval, 自建测试 |
